   ```dart
   Future<void> _importFromClipboard(BuildContext context, WidgetRef ref) async {
     final l10n = AppLocalizations.of(context)!;
     final text = await ClipboardHelper.getText();
     if (text == null || text.isEmpty) {
       ScaffoldMessenger.of(context).showSnackBar(
         SnackBar(content: Text(l10n.parseErrorEmptyClipboard), duration: AppConstants.snackBarDurationLong),
       );
       return;
     }
     final config = ShareLinkParser.parse(text);
     if (config == null) {
       ScaffoldMessenger.of(context).showSnackBar(
         SnackBar(
           content: Text(l10n.parseErrorInvalidLink),
           duration: AppConstants.snackBarDurationLong,
           action: SnackBarAction(label: l10n.retryAction, onPressed: () => _importFromClipboard(context, ref)),
         ),
       );
       return;
     }
     // Check for duplicates by address+port+protocol
     final servers = await ref.read(serverListNotifierProvider.future);
     final isDuplicate = servers.any((s) => s.address == config.address && s.port == config.port && s.protocol == config.protocol);
     if (isDuplicate) {
       ScaffoldMessenger.of(context).showSnackBar(
         SnackBar(content: Text(l10n.duplicateServer), duration: AppConstants.snackBarDurationDefault),
       );
       return;
     }
     await ref.read(serverListNotifierProvider.notifier).addServer(config);
     ScaffoldMessenger.of(context).showSnackBar(
       SnackBar(
         content: Text('${l10n.importSuccess} — ${config.name}'),
         duration: AppConstants.snackBarDurationDefault,
         backgroundColor: Colors.green.shade700,
         action: SnackBarAction(label: l10n.viewAction, onPressed: () {}),
       ),
     );
   }
   ```

## Trust Boundaries

| Boundary | Description |

| Clipboard → app | User clipboard content is untrusted input, could contain malformed or malicious strings |
| Paste dialog → parser | Free-text user input parsed as share link or JSON |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-05-01 | Tampering | import_fab.dart (clipboard) | mitigate | Clipboard content treated as untrusted: passed through ShareLinkParser.parse which validates format, wraps in try-catch, returns null on failure. No raw clipboard data displayed without parsing. |
| T-01-05-02 | Tampering | paste_config_dialog.dart | mitigate | Free-text input validated by ShareLinkParser.parse before any storage. Invalid input shows inline error, never stored. Server names truncated to 50 chars (AppConstants.maxServerNameLength). |
| T-01-05-03 | Denial of Service | paste_config_dialog.dart | mitigate | ShareLinkParser rejects input > 10000 chars (from Plan 03). TextField does not have a maxLength set (to allow JSON paste) but the parser enforces limits. |
| T-01-05-04 | Spoofing | server_card.dart | accept | Server names come from share link fragment field — they could contain misleading text (e.g., "Free USA Server" that's actually in Russia). This is inherent to the proxy config domain. Users are expected to verify their own configs. No mitigation needed at L1. |
