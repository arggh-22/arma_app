**1. Create `lib/features/server/data/parsers/sip008_parser.dart`:**
- Class `Sip008Parser` with static `List<ServerConfig>? tryParse(String body)`
- Detect two formats: direct JSON array (starts with `[`) or wrapped `{"version": N, "servers": [...]}`
- Map fields: `server` → address, `server_port` → port, `password` → password, `method` → method, `remarks` or `tag` → name
- All parsed servers get `protocol: ProtocolType.shadowsocks`
- Generate UUID v4 for each server's id, set `addedAt: DateTime.now()`
- Wrap `jsonDecode` in try-catch, return null on parse failure

**2. Create `lib/features/server/data/parsers/clash_parser.dart`:**
- Class `ClashParser` with static `List<ServerConfig>? tryParse(String yamlContent)`
- Uses `loadYaml()` from `package:yaml` to parse
- Extract `proxies:` list from YamlMap
- Map Clash type names: `vmess` → ProtocolType.vmess, `vless` → ProtocolType.vless, `trojan` → ProtocolType.trojan, `ss` → ProtocolType.shadowsocks, `hysteria2`/`hy2` → ProtocolType.hysteria2
- Skip unrecognized types (return null for that entry)
- Map nested transport options:
  - `ws-opts.path` → ServerConfig.path
  - `ws-opts.headers.Host` → ServerConfig.host
  - `grpc-opts.grpc-service-name` → ServerConfig.serviceName
  - `h2-opts.path` → ServerConfig.path
  - `h2-opts.host` → ServerConfig.host (first entry if list)
- Map `tls: true` → security: 'tls', `tls: false` or absent → security: 'none'
- Map `sni` or `servername` → ServerConfig.sni
- `cipher` → ServerConfig.method (for SS) or ServerConfig.encryption (for VMess)
- `uuid` → ServerConfig.uuid, `password` → ServerConfig.password

**3. Create `lib/features/server/data/parsers/subscription_parser.dart`:**
- Class `SubscriptionParser` with static `List<ServerConfig> parseBody(String body)`
- Format detection order (per D-01, D-02):
  1. If body trimmed starts with `[` or `{` → try `Sip008Parser.tryParse()`, if non-null non-empty return it
  2. If body contains `proxies:` → try `ClashParser.tryParse()`, if non-null non-empty return it
  3. Try base64 decode: normalize padding (`while (s.length % 4 != 0) s += '='`), replace URL-safe chars (`-` → `+`, `_` → `/`), then `utf8.decode(base64Decode(normalized))`. If decode succeeds, use decoded text. If fails, use original body as plain text.
  4. Split resulting text by `\r\n` or `\n`, trim each line, filter empty, call `ShareLinkParser.parse(line)` on each, collect non-null results
- Return the list of parsed ServerConfigs (may be empty if all lines are invalid)

**1. Create `lib/features/server/data/parsers/share_link_generator.dart`:**
- Class `ShareLinkGenerator` with static `String generate(ServerConfig server)`
- Dart 3 switch expression on `server.protocol` dispatching to private methods:

**VLESS** (`_generateVlessLink`):
- Format: `vless://uuid@address:port?params#name`
- Params: `type` (network), `security`, `sni` (if set), `host` (if set), `path` (if set, URI-encoded), `flow` (if non-empty), `fp` (fingerprint), `pbk` (publicKey), `sid` (shortId), `spx` (spiderX, URI-encoded), `serviceName` (if gRPC), `alpn` (if set)
- Fragment: URI-encoded server name

**VMess** (`_generateVmessLink`):
- Format: `vmess://base64(JSON)` — legacy format for maximum compatibility (per Pitfall 8)
- JSON fields: `v: "2"`, `ps: name`, `add: address`, `port: port.toString()`, `id: uuid`, `aid: alterId.toString()`, `net: network`, `type: "none"`, `host: host ?? ""`, `path: path ?? ""`, `tls: security == 'tls' ? 'tls' : ''`, `sni: sni ?? ""`, `fp: fingerprint ?? ""`
- Use `base64Encode(utf8.encode(jsonEncode(json)))` — no line breaks in base64

**Trojan** (`_generateTrojanLink`):
- Format: `trojan://password@address:port?params#name`
- Params: `type` (network), `security`, `sni`, `host`, `path` (URI-encoded), `serviceName` (if gRPC), `fp`, `alpn`

**Shadowsocks** (`_generateSsLink`):
- Format: `ss://base64(method:password)@address:port#name`
- The method:password part is base64-encoded (standard SIP002 format)
- Use `base64Encode(utf8.encode('${method}:${password}'))` — no padding issues (standard base64)

**Hysteria2** (`_generateHy2Link`):
- Format: `hysteria2://password@address:port?params#name`
- Params: `sni`, `obfs` (if set), `obfs-password` (if set), `insecure` (if applicable)
