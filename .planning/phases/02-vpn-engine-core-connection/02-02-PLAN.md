**Create `lib/xray/xray_config_builder.dart`:**

Import `dart:convert` for `jsonEncode`, import `ServerConfig` and `ProtocolType`.

Create class `XrayConfigBuilder` with private constructor and a single public static method:

```dart
static String build(ServerConfig server)
```

This method returns `jsonEncode(config)` where `config` is a `Map<String, dynamic>` with these top-level keys:
- `"log"`: `{"loglevel": "warning"}`
- `"stats"`: `{}` — CRITICAL: must be present for QueryStats to work
- `"policy"`: `_buildPolicy()` — enables traffic stats collection
- `"dns"`: `_buildDns()` — split DNS per D-11
- `"inbounds"`: `[_buildSocksInbound()]` — SOCKS5 proxy for TUN
- `"outbounds"`: `[_buildProxyOutbound(server), _buildDirectOutbound(), _buildBlockOutbound()]`
- `"routing"`: `_buildRouting()` — LAN bypass per D-12

**Private helper methods (all return `Map<String, dynamic>`):**

`_buildPolicy()`:
```dart
{
  'levels': {'0': {'statsUserUplink': true, 'statsUserDownlink': true}},
  'system': {'statsOutboundUplink': true, 'statsOutboundDownlink': true},
}
```

`_buildDns()`:
```dart
{
  'servers': [
    {'address': '1.1.1.1', 'domains': [], 'port': 53},
    'localhost',
  ],
}
```

`_buildSocksInbound()`: SOCKS5 inbound on 127.0.0.1:10808 with `"auth":"noauth"`, `"udp":true`, `"userLevel":0`, sniffing enabled with `destOverride: ["http", "tls", "quic"]`, `metadataOnly: false`, `routeOnly: false`. Tag: `"socks-in"`.

`_buildProxyOutbound(server)`: Tag `"proxy"`, protocol from `server.protocol.scheme` (map `ss` → `shadowsocks`), settings from `_buildProtocolSettings(server)`, streamSettings from `_buildStreamSettings(server)`.

`_buildProtocolSettings(server)` — use Dart 3 switch expression on `server.protocol`:
- `ProtocolType.vless` → `vnext` with users `id: server.uuid, encryption: "none", flow: _resolveFlow(server)`
- `ProtocolType.vmess` → `vnext` with users `id: server.uuid, alterId: server.alterId, security: (server.encryption == 'none' ? 'auto' : server.encryption)`
- `ProtocolType.trojan` → `servers` with `address, port, password: server.password`
- `ProtocolType.shadowsocks` → `servers` with `address, port, method: server.method, password: server.password`
- `ProtocolType.hysteria2` → `servers` with `address, port, password: server.password` (placeholder for Phase 4)

`_resolveFlow(server)`:
- Return empty string `''` if protocol is NOT vless
- Return empty string `''` if network is NOT `tcp`
- Return empty string `''` if security is NOT `tls` and NOT `reality`
- Otherwise return `server.flow ?? ''`

`_buildStreamSettings(server)`:
- `"network"`: `server.network`
- `"security"`: `server.security`
- If security == "tls": add `"tlsSettings"` with `serverName: server.sni ?? server.address`, `allowInsecure: false`, `alpn: server.alpn?.split(',') ?? []`, `fingerprint: server.fingerprint ?? 'chrome'`
- If security == "reality": add `"realitySettings"` with `serverName: server.sni ?? server.address`, `fingerprint: server.fingerprint ?? 'chrome'`, `publicKey: server.publicKey ?? ''`, `shortId: server.shortId ?? ''`, `spiderX: server.spiderX ?? ''`
- Add transport-specific settings based on `server.network`:
  - `"tcp"` → `"tcpSettings": {"header": {"type": "none"}}`
  - `"ws"` → `"wsSettings": {"path": server.path ?? "/", "headers": {"Host": server.host ?? server.address}}`
  - `"grpc"` → `"grpcSettings": {"serviceName": server.serviceName ?? "", "authority": server.authority ?? "", "multiMode": false}`
  - `"h2"` → `"httpSettings": {"host": [server.host ?? server.address], "path": server.path ?? "/"}` — note: H2 always forces security to "tls"

`_buildDirectOutbound()`: `{tag: "direct", protocol: "freedom", settings: {}}`
`_buildBlockOutbound()`: `{tag: "block", protocol: "blackhole", settings: {response: {type: "http"}}}`

`_buildRouting()`:
```dart
{
  'domainStrategy': 'IPIfNonMatch',
  'rules': [
    {'type': 'field', 'outboundTag': 'direct', 'ip': ['geoip:private']},
    {'type': 'field', 'outboundTag': 'direct', 'domain': ['geosite:private']},
    {'type': 'field', 'outboundTag': 'proxy', 'port': '0-65535'},
  ],
}
```
