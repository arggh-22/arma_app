## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Tight inline gaps (icon/text micro-spacing) |
| sm | 8px | Between related controls in same step card |
| md | 16px | Default paddings, standard vertical rhythm |
| lg | 24px | Between major sections (step groups, form/action block) |
| xl | 32px | Top breathing space under app bar / section starts |
| 2xl | 48px | Large section separation on tall layouts |
| 3xl | 64px | Reserved page-level separation (rare in this screen) |

Exceptions: 12px card inner padding and 12px corner radius are allowed to match existing dashboard card pattern; minimum touch target remains 48px.

---

## Visual Hierarchy Contract

Guide screen visual priority order:
1. Primary focus: `Open Telegram Bot` CTA at the top.
2. Secondary focus: step cards explaining Start → ID retrieval (`Get Telegram ID` or `/my_id`).
3. Tertiary focus: Telegram ID input + `Paste` + primary `Link` submit actions.

This order is mandatory and should remain visually obvious through spacing, grouping, and contrast.

---

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | `Link` |
| Empty state heading | `No Telegram ID yet` |
| Empty state body | `Open the bot, run Get Telegram ID or /my_id, then paste your ID to continue.` |
| Error state | `Couldn’t link Telegram right now. Check your ID and connection, then try again.` |
| Destructive confirmation | none in Phase 12 |

Outcome-specific feedback contract:
- `linked`: `Telegram account linked successfully.` → auto-return to Dashboard.
- `unauthorized`: `Session expired. Please sign in again, then retry linking.` → stay on screen.
- `alreadyLinked`: `This Telegram account is already linked.` → stay.
- `invalidId`: `Telegram ID is invalid. Paste numeric ID (5–20 digits).` → stay.
- `network`: `No network connection. Reconnect and try again.` → stay.
- `server`: `Server error. Please try again shortly.` → stay.
- `unknown`: `Unexpected error. Please try again.` → stay.
