## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Badge vertical padding, micro gaps |
| sm | 8px | Tight row gaps, small separators |
| md | 16px | Standard card/screen padding |
| lg | 24px | Section separation in dashboard flow |
| xl | 32px | Major group spacing when needed |
| 2xl | 48px | Large visual breaks |
| 3xl | 64px | Rare page-level breathing room |

Exceptions: 12px card radius/internal tile padding; 120px connect button diameter; 56px FAB touch target.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | ColorScheme.surface (light/dark theme surface) | Scaffold/background surfaces |
| Secondary (30%) | ColorScheme.surfaceContainerLow | Cards, grouped containers, parked server surface |
